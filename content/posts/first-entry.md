This is the first entry in my digital garden. A place to collect thoughts, notes, and things I learn along the way.

## What is this?

A personal knowledge base built with [Hugo](https://gohugo.io/) and the [Mana](https://github.com/Livour/hugo-mana-theme) theme. Content is organized into posts with tags for easy filtering.

## How it works

Content is written in Markdown. Push to GitHub and it deploys automatically via GitHub Actions.
